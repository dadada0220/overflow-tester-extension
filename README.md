# overflow-tester

UIの堅牢性チェック用 Chrome 拡張機能。コーディング完了時やレビュー時に、テキストが増えても・繰り返し要素が増えても崩れないかをワンクリックで確認できます。

## スクリーンショット

ポップアップ（幅280px）からテキスト膨張・要素複製の倍率を設定し「Run」を押すだけです。元に戻すにはページをリロードします。

---

## 機能

### テキスト膨張

ページ内の表示テキストノードを、現在の文字列の N 倍に置換します。

- **例：** 「送信する」→「送信する送信する送信する」（3倍の場合）
- `<script>` / `<style>` / `<title>` / `<head>` 配下は除外
- 空白のみのテキストノードは除外

### 要素複製

親要素直下に同じタグまたは同じクラス名の要素が 2 つ以上ある場合、それを「繰り返し要素」と判定して N 倍に複製します。

- **例：** `<li>` が 3つ → 9つ（3倍の場合）
- 最大グループのみ複製（爆発的増殖防止）
- タグ名 → クラス名の順で判定し、より要素数の多いグループを優先

### 除外セレクタ

指定したセレクタにマッチする要素とその子孫を、テキスト膨張・要素複製の両方から除外します。

- カンマ区切りで複数指定可能
- **例：** `header, #sidebar, .skip-test`
- 無効なセレクタは無視されます（エラーになりません）

---

## インストール

ビルド済みの拡張機能は `dist/` フォルダに格納されています。

1. `dist/` フォルダを任意の場所に置く
2. Chrome で `chrome://extensions/` を開く
3. 右上の「**デベロッパーモード**」を ON にする
4. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
5. `dist/` フォルダを選択

## 開発

### 技術スタック

| 項目 | 採用技術 |
|------|---------|
| ビルド | Vite + React |
| UI | shadcn/ui（Switch / Label / Button / Separator） |
| スタイル | Tailwind CSS v3 |
| アイコン | lucide-react |
| Manifest | V3 |

### ディレクトリ構成

```
overflow-tester/
├── src/
│   ├── App.jsx              # ポップアップUI + ページ注入関数
│   ├── main.jsx             # Reactエントリーポイント
│   ├── index.css            # Tailwind + shadcn CSS変数
│   ├── lib/
│   │   └── utils.js         # cn() ユーティリティ
│   └── components/ui/
│       ├── button.jsx
│       ├── label.jsx
│       ├── separator.jsx
│       └── switch.jsx
├── dist/                    # ビルド出力（拡張機能として読み込むフォルダ）
│   ├── manifest.json
│   ├── index.html
│   ├── assets/
│   └── icons/
├── index.html
├── vite.config.js
├── tailwind.config.js
└── package.json
```

### セットアップ

```bash
npm install
```

### 開発ビルド（ウォッチ）

```bash
npm run dev
```

※ Chrome拡張機能はファイルプロトコルで動作するため、`npm run dev` のプレビューサーバーでは動作しません。変更のたびに `npm run build` → Chrome の拡張機能ページで「更新」ボタンを押してください。

### プロダクションビルド

```bash
npm run build
```

`dist/` が更新されます。Chrome の `chrome://extensions/` で当該拡張機能の「更新」ボタンを押すと反映されます。

---

## 使い方

1. 確認したいページを Chrome で開く
2. 拡張機能アイコンをクリックしてポップアップを開く
3. テキスト膨張・要素複製の ON/OFF と倍率を設定
4. 必要に応じて除外セレクタを入力
5. **Run** を押す
6. レイアウト崩れがないか確認
7. リロードで元に戻す

---

## 注意事項

- `chrome://` や `edge://` などのブラウザ内部ページでは動作しません
- 復元機能はありません。元に戻すにはページをリロードしてください
- SPAなどでページ内容を動的に切り替えた場合は、再度 Run を実行してください
